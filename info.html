Who am I?

Our pensum covered programming, IT and a little bit of Networking. We learned Java with Hibernate, C# with ASP.NET and some JavaScript. Also Database Structure with SQL.

I learned a lot with this Full-Stack! Everything was mocked up with Jade (currently known as Pug), with a CSS pre-processor called Less.

In my free time I like to read about new technologies in the Web Development area, do experiments (like this page!) with everything I find on the web and learn new languages or frameworks (I'm currently learning ReactJS, VueJS and I got a lot of interest in Ruby.)
